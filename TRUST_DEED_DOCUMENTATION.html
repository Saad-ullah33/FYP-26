01

Executive Overview

The TrustDeed Verification System is a cryptographically secured property title deed verification engine integrated into NextProperty (FYP-26). It allows buyers, sellers, real estate investors, land registry officials, and public users to verify the authenticity, ownership integrity, and transaction parameters of real estate properties — without relying on centralized physical paperwork.

🏛️

Architecture Model: Self-contained client-side cryptographic service layer (deedService.js) using deterministic HMAC signature simulation, local storage persistence (fyp26_trust_deeds), GDPR-compliant role-based masking, and 1-click verification UI (DeedVerification.jsx).

02

Core Functionalities

🔐 Cryptographic HMAC Signature Generation

Deterministic Base64 payload hashing using central secret keying. Ensures URL parameters cannot be tampered with — any modification immediately triggers a TAMPERING DETECTED alert.

Signature = Base64( deedId + "-" + role + "-" + SECRET_KEY ) → 32-char hex string
🛡️ GDPR-Compliant Role-Based Access & Data Masking

The system dynamically controls data visibility based on the authenticated URL role parameter:

public

Masks CNICs, emails & full names. Safe for any anonymous scan.

buyer

Unmasked full transaction data for the purchasing party.

seller

Full ownership records for the property's selling party.

admin

Full audit logs, registry authority proofs, and ledger detail.

🏛️ TrustDeed Certified Registry Vault

Pre-seeded real estate deeds across major metropolitan locations (Lahore, Islamabad, Faisalabad, Rawalpindi). Multi-criteria search by Deed ID, title, city, or party name. 1-click cryptographic verification for any registered property.

Print-optimized CSS (@media print) renders a formal Digital Title Deed Certificate with double gold borders, official seals, signature lines, and full transaction details.

05

Developer API Reference

generateSignature(deedId, role)

Generates a 32-character hex-like signature string for a given deedId and role.

JavaScript 
import { generateSignature } from "../utils/deedService";

const sig = generateSignature("TD-101-9921", "public");
// Returns: "d839e1a02837bcde8f12a34b56cd7e8a"
verifyDeedSignature(deedId, role, sig)

Verifies the cryptographic signature and returns the deed object with GDPR masking applied for public role.

JavaScript 
const result = verifyDeedSignature("TD-101-9921", "public", sig);

// Returns:
// {
//   verified: true,
//   role: "public",
//   deed: {
//     deedId: "TD-101-9921",
//     buyerName: "A** Khan",    ← GDPR masked
//     buyerCNIC: "374**-****-9" ← GDPR masked
//   }
// }
createDeed(deedData)

Registers a new deed entry into local storage and auto-generates a unique deedId & blockchain hash.

JavaScript 
const newDeed = createDeed({
  propertyId: 204,
  title:      "10 Marla Luxury House",
  location:   "DHA Phase 5, Islamabad",
  price:      "3.2 Crore PKR",
  buyerName:  "Hamza Ali",
  sellerName: "Usman Raza",
});
getQRBaseUrl()

Resolves the correct base URL for QR code generation — supports Vercel production, local overrides, and environment variables.

JavaScript 
const base = getQRBaseUrl();
// Returns: "https://nextpropertypk.vercel.app"
// (or current window origin when deployed)

Summary of Enhancements

  • Self-Contained Architecture — TrustDeed operates entirely on client-side cryptographic services with no external API dependency.
  • Navigation Integration — Streamlined Navbar header with TrustDeed verification in "More" dropdown and Footer quick links.
  • Certified Vault Landing View — Interactive deed vault with city filters, live search, and 1-click verification cards.
  • GDPR Public Masking Enforced — Public scans mask all sensitive CNICs, phone numbers, and email addresses.
  • User Dashboard Integration — "Digital Deeds" tab in UserDashboard.jsx for deed management and verification requests.
  • Admin Dashboard Vault & Demo Generator — Full TrustDeed administration vault with 1-click Demo Deed Generator and SHA-256 Title Approval.
  • In-App Camera QR Scanner — Built-in webcam & image QR scanner powered by html5-qrcode on /verify-deed.
  • Vercel SPA Routing — vercel.json rewrites ensure all deep-link paths load correctly on mobile browsers without 404 errors.
  • Production Verified — Build confirmed via npx vite build with zero errors, deployed on Vercel at nextpropertypk.vercel.app.
